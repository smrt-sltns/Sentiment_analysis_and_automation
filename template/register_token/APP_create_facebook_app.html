Description

The software requires you to set up a Facebook developers account and create a Graph API APP.

Follow the below steps to store APP ID and APP SECRET.

  1. Go to Developers dashboard and click on 'Create App'.
  2. Name your app as "Sentiment APP" and give all the necessary
    permission to create the app.
  3. Go back to dashboard, Under "Admin Apps" click on "Sentiment APP".
  4. On left navigation find Settings >> Basic
  5. Click on "Show" of App Secret and reveal the Secret by providing Facebook password.
  6. Copy App Id and App Secret in a text file.
  7. Let's move forward!
