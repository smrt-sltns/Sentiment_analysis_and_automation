Save Access Token

  1. Go to Tool Explorer of developers account select "Sentiment App" in "Meta App".
  2. Scroll bottom of the right hand side navigation and find the form saying "Add a Permission".
  3. Add all the permission from below list one by one to 'Add a Permission'.
    ['pages_manage_instant_articles', 'pages_show_list', 'ads_management', 'ads_read', 'business_management', 'leads_retrieval', 'pages_read_engagement', 'pages_manage_metadata', 'pages_read_user_content', 'pages_manage_ads', 'pages_manage_posts', 'pages_manage_engagement' ]
  4. Cross check if all the permissions listed above match with Access token permissions.
  5. Click on button 'Generate Access Token' and agree to all Facebook permissions.
    6. **If you miss out on any of the permissions please add in the permission and click on "Generate Access Token" again.
